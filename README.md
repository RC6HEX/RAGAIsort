# Neural Library - RAG система для работы с книгами

Интеллектуальная система для анализа текстовых документов с использованием Gemini 2.5 Flash API.

## Стек технологий

- **Frontend**: React + TypeScript + Vite
- **UI**: TailwindCSS + Motion (Framer Motion)
- **AI**: Google Gemini 2.5 Flash API
- **База данных**: IndexedDB (локальное хранилище)
- **Векторный поиск**: Простой текстовый поиск с ранжированием

## Возможности

### Обработка документов
- Загрузка .txt файлов
- Автоматическое разбиение на чанки с перекрытием (overlap)
- Отклонение пустых файлов
- Индексация с метаданными (имя файла, номер фрагмента)

### Два режима работы

#### 1. Режим "Поиск"
Начните запрос со слов "найди" или "поиск"

**Пример:**
```
Найди информацию о Наташе Ростовой
```

**Результат:** Список из 3-5 самых релевантных фрагментов с указанием источника

#### 2. Режим "Вопрос"
Задайте обычный вопрос

**Пример:**
```
Кто такая Наташа Ростова?
```

**Результат:** Развернутый ответ на основе контекста с цитатами из текста

### Дополнительные функции
- Автоматическая генерация 4 вопросов по содержанию каждой книги
- Сохранение истории чатов и настроек
- Персонализированные ответы (обращение по имени)
- Темная тема с настраиваемыми цветовыми схемами

## Установка и запуск

### Требования
- Node.js 18+
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Настройка API ключа

API ключ Gemini уже встроен в код (`src/gemini.ts`). Для использования собственного ключа:

1. Получите ключ на https://makersuite.google.com/app/apikey
2. Замените значение `API_KEY` в файле `src/gemini.ts`

### Запуск dev-сервера

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

### Сборка для production

```bash
npm run build
```

## Примеры использования

### Загрузка книг
1. Нажмите на область загрузки или перетащите .txt файлы
2. Дождитесь завершения индексации (статус "В базе")

### Режим поиска
```
Найди все упоминания о войне
Поиск информации о главных героях
```

### Режим вопросов
```
Кто главный герой книги?
Какие события происходят в первой главе?
Опиши характер Рыжика
```

### Работа с конкретной книгой
1. Нажмите правой кнопкой на книгу в списке
2. Выберите "Обсудить книгу"
3. Задавайте вопросы по этой книге

## Архитектура

```
src/
├── App.tsx              # Главный компонент
├── gemini.ts            # API Gemini (генерация ответов, вопросов)
├── chunking.ts          # Разбиение текста на чанки
├── vectorStore.ts       # Хранилище и поиск по чанкам
├── db.ts                # IndexedDB (сохранение данных)
└── index.css            # Стили
```

## Особенности реализации

### Chunking
- Размер чанка: 1000 символов
- Перекрытие (overlap): 200 символов
- Метаданные: имя книги, индекс чанка, позиция в тексте

### Поиск
- Простой текстовый поиск с подсчетом совпадений
- Бонус за точное совпадение фразы
- Возврат топ-5 релевантных фрагментов

### Обработка ошибок
- Проверка на пустые файлы
- Обработка ошибок API
- Информативные сообщения об ошибках
